# Resume Download Setup

## How to Add Your Resume

1. **Place your resume PDF** in the `/public` directory
2. **Name it one of these** (the component will try these in order):
   - `Mehul_Shekhada_Resume.pdf` (recommended)
   - `resume.pdf`
   - `Mehul-Shekhada-Resume.pdf`
   - `CV.pdf`

3. **The download buttons are already set up** in:
   - Footer (icon button)
   - About page (full button next to title)
   - Contact page (link in social section)

## File Location

```
/public/Mehul_Shekhada_Resume.pdf
```

Once you add the file, the download buttons will automatically work!

## Testing

After adding the file:
1. Refresh your browser
2. Click any "Download Resume" button
3. The PDF should download automatically
